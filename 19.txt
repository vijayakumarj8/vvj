19. Write a python programming to perform data transfer operation in arduino. 
  

Here is an example Python program that performs data transfer operations with Arduino using the PySerial library:


import serial

# Open the serial port
ser = serial.Serial('COM3', 9600)  # Replace COM3 with your Arduino's serial port

# Send data to Arduino
data = "Hello, Arduino!"
ser.write(data.encode())

# Receive data from Arduino
received_data = ser.readline().decode()
print("Received from Arduino:", received_data)

# Close the serial port
ser.close()



Arduino Code:


void setup() {
  Serial.begin(9600);
}

void loop() {
  if (Serial.available()) {
    String data = Serial.readString();
    Serial.println("Received from Python: " + data);
    Serial.println("Hello, Python!");
  }
}
